## 🚀 Guía de Carga a GitHub (Protocolo Seguro)
Si recibes el error `failed to push some refs`, ejecuta estos comandos en orden:

1. `git init` (Si no lo has hecho)
2. `git remote add origin https://github.com/umbralcero7-beep/Aurora.git` (Si no está vinculado)
3. `git add .`
4. `git commit -m "Aurora OS V9.5 - Milestone de Patentado"`
5. `git pull origin main --rebase` (Resuelve el conflicto con archivos remotos)
6. `git push origin main` (Despacha el código a GitHub)

## 🛠️ Núcleo Tecnológico (Stack de Ultra-Respuesta)
- **Frontend**: Next.js 15 (App Router) con arquitectura Serverless.
- **Cerebro**: Google Gemini 2.0 Flash via Genkit (Auditoría en tiempo real).
- **Base de Datos**: Firebase Firestore con aislamiento Multi-sede y **Modo Offline Real**.
- **ERP**: Modelado financiero inspirado en Microsoft Dynamics y Oracle NetSuite.
