# **App Name**: Aurora Inventory AI

## Core Features:

- Inventory Management: Comprehensive control of products, stock levels, and inventory movements.
- Cost Calculation: Cost calculation using FIFO, LIFO, and weighted average methods.
- Demand Prediction: AI-powered prediction of demand to optimize inventory levels and reduce costs. The LLM will use current and historic stock and sales level data to decide what the expected future stock levels should be, acting as a tool for purchase managers.
- Anomaly Detection: Detection of unusual patterns in sales data to identify potential fraud or errors, alerting administrators. The LLM acts as a tool.
- Advanced Reporting: Generation of advanced reports with charts and export options to Excel and Google Sheets.
- User Authentication and Roles: JWT authentication system with multiple roles (ADMIN, SUPERVISOR, INVENTORY, AUDITOR, CASHIER) to manage user permissions.
- Auditing: Comprehensive logging of all system actions in MongoDB for auditing and security purposes.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#008080) to highlight key actions and information.
- Body and headline font: 'Inter', a sans-serif, for a modern and neutral look suitable for both headlines and body text.
- Use clear, professional icons representing inventory, costs, reports, and other key features.
- Clean, well-organized layout with intuitive navigation to facilitate efficient inventory management.
- Subtle animations for transitions and feedback, enhancing user experience without being distracting.
